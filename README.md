# pedido-facil
O sistema gerencia pedidos de um restaurante, permitindo listar produtos, registrar pedidos por mesa e notificar a cozinha e o garçom sobre o status dos pedidos. A cozinha atualiza o status para "em preparação" e "pronto", enquanto o garçom marca como "entregue", removendo o pedido da fila após a entrega, em uma interface em tempo real.
